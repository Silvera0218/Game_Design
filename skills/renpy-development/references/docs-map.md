# Ren'Py 繁中官方文档完整路由

## 版本说明

- 来源：<https://doc.renpy.cn/zh-TW/>
- 建档时站点版本：Ren'Py 8.4.2
- 索引规模：103 个文档页面，已全部纳入下表。
- Ren'Py 8.5.3 英文文档相较此索引新增 `testcases`、`display_problems`，并不再列出 `modes`、`distributor`。
- 精确 API、弃用、平台要求和安全行为必须按项目目标版本复核。
